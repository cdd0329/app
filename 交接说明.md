1. `flutter clean` + `flutter pub get` ❌
2. 删除 `build/` + `.dart_tool/` 目录 ❌
3. 杀掉 Kotlin daemon ❌
4. 删除 `.gradle/caches/` ❌
5. 配置 Gradle 代理（阿里云镜像/HTTP_PROXY）❌

### 需解决
1. **Kotlin 增量编译缓存损坏** — 可能是 Gradle 版本冲突或 Kotlin 版本不一致

2. **Flutter SDK artifacts 下载** — 需要能访问 `storage.googleapis.com`（开全局 VPN 或代理）

3. **Gradle 依赖下载** — 需要能访问 Maven Central（已通过手动配置 gradle.properties 解决）
